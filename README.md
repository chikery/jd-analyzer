# JD Analyzer

채용공고 URL을 넣으면 요구 역량을 분석하고, 내 스킬셋과 매칭 점수를 알려주는 도구.

## 기능 (v1 예정)
- 채용공고 URL 입력
- 요구 역량 자동 추출
- 내 스킬셋과 매칭 점수 계산
- 부족 역량 리포트

## 사용 기술
- Python 3.10+
- Google Gemini API (gemini-2.5-flash-lite)
- BeautifulSoup (웹 스크래핑)
- Streamlit (UI, v3 예정)
